## [1.17.0] – Senses 3 Preparation
**Summary:** Prepared the asset for upcoming Senses 3 integration while maintaining full compatibility with Senses 2. This update also improves GameObject destruction flexibility and overall asset stability.

### Added
- **Senses 3 Preparation**
  - Asset prepared for **Senses 3**
  - Continues to work fully with **Senses 2**

- **Node: GameObject Destroy – Target Selection**
  - Node now allows choosing **which GameObject** will be destroyed
  - Target can be selected from:
    - **Soft Blackboard reference**
    - **Hardcoded Blackboard reference**
  - Enables more flexible and reusable Behaviour Tree setups

### Changed
- Internal improvements focused on smoother and more predictable asset usage
- Minor behavior refinements across nodes and editor workflows

### Fixed
- Multiple stability-related fixes in both editor and runtime
- Improved reliability during node execution and Behaviour Tree evaluation
